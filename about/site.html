零、准备工作(基础版)

  • 该网站是基于 hexo 使用 yun 主题部署在 github 的博客
  • 参见 hexo 官方文档 ，在按要求装好hexo、Node.js、Git后，即可愉快的搭建自己的网站啦~
  • 如果想获得更好的体验，建议使用hexo主题 。
  • 如果你想使用跟我一样的主题，请参照 https://yun.yunyoujun.cn/ ，你的99%的问题都能在这里找到答案( •̀ ω •́ )y
  • 对于小白，搭建好一个框架大概花费2天时间（保守估计）
